# wangy-wangy-tool

wangy wangy tool adalah sebuah aplikasi sederhana yang digunakan untuk membuat copypasta secara instan tanpa harus ngedit lagi, untuk template insya allah akan ditambah dengan seiring waktu.

# fitur 

- mobile friendly
- copy langsung
- gampang digunakan

# demo
